# Privacy Policy

**dev-journal** — Claude Code Plugin

## Overview

The dev-journal plugin runs entirely locally within Claude Code. It has no backend, no external dependencies, and collects no data of any kind.

## Data Collection

- **No telemetry** — the plugin does not collect usage data, analytics, or crash reports.
- **No tracking** — there are no cookies, identifiers, or fingerprinting mechanisms.
- **No third-party services** — the plugin does not communicate with any external APIs or services.

## Data Storage

Journal entries are stored exclusively in your local project directory (typically `docs/journal/`). The plugin does not write data anywhere outside the project.

## Network Activity

The plugin itself transmits no data over the network. Any network activity that occurs during use is handled by Claude Code itself and is subject to [Anthropic's privacy policy](https://www.anthropic.com/privacy).
